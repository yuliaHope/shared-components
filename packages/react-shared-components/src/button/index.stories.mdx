### Props

<Props of={Button} />

##### The component accepts all other props [`style`, `onClick`, etc]

<br />


When `icon` property is provided as string it renders an SVG Icon
(referenced by name from `@hulu-react-style-components/icons`) along with the specified children.

### Usage

```jsx
import { Button }  from '@hulu/react-style-components';


<Button onClick={onButtonClick} size={size} variant={variant} disabled={disabled}>Button text</Button>
```
